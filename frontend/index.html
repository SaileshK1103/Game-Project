Vaccine Quest Game

Your mission is to fly to different airports to collect essential elements (A, B, C, D) needed to create a vaccine.

Each element has a limited number available in the game.

A random airport will also contain a lucky box, which costs $100 to open.

The lucky box may contain an additional element or be empty.

If you collect all required elements within the budget provided at the start of the game, you win!
